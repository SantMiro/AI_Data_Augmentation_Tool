Data Augmentation Tool



This Data Augmentation Tool takes your imbalanced dataset (95/5) of shape (n,29) and returns it at a 60/40 proportion.

The tool pipeline starts with a novel Generative Adversarial Network (GAN), that generates the number of samples needed to balance the dataset. The second step is to add noise to the dataset and then perform a scaling of the features.
